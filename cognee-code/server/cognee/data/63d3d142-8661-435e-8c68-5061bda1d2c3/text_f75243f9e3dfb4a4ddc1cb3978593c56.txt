# 关键帧推进

[0s-4s]
镜头从远处缓缓推近，@图片1 方舟都市全貌充满画面：最底层飞行火车骨架、其上密集多层彩色积木建筑、右侧黄色吊臂吊车、顶端金属圆球气囊、左侧小型双翼飞机绕飞依次入镜，烟囱群持续冒白烟，阳光从右侧云层透出打亮金色轮廓，小人偶在各层阳台活动。

镜头随即切入 @图片2，初始飞艇悬停于低空废墟上方：顶部彩色椭球气囊砖块拼缝、黄铜链条、左侧花形小螺旋桨逐一清晰入镜；小人偶倚舱门眺望远方，橙色卡扣拼缝依次亮起。
