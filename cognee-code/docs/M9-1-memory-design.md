1. 为什么需要 Reflector？

归纳：把“用户昨天买了苹果，今天买了香蕉”转化成“用户有吃水果的习惯”。

纠偏：如果 Agent 刚才的表现让用户生气了，Reflector 会记录下：“下次不要使用这种反问语气”。

思考：这些对话里有新信息吗？有矛盾的信息吗？有什么值得以后学习的经验吗？

多维过滤 (Metadata)：

针对 Category: Preference 的记录给予更高的权重。

重排序 (Rerank)：根据 Recency（新旧程度）对结果排序。

检索 Pipeline：

重排序 (Reranking)：
